项目负责人教程：先准备两样资料

当前项目 的负责人照着下面做即可，不需要改代码，也不要把 ChatDaddy 登录密码贴进来。

1

建立 API Token

打开 ChatDaddy → Developer → API Token → +Create。名称填写 ReplyPilot-项目名，发送权限选择 MESSAGES_SEND_TO_ALL，保存后复制 Token。

2

复制 Channel / Account ID

打开 ChatDaddy 右上角账号菜单 → Manage Channels，找到这个项目使用的 WhatsApp Channel，然后复制它的 Account ID。

3

贴进下面两个空格

先把 Account ID 贴进 ChatDaddy Channel / Account ID，再把 Token 贴进 ChatDaddy API Token。Mac 按 ⌘V，Windows 按 Ctrl+V。

4

连接并同步

点击 连接并同步 Flow。看到“ChatDaddy 已连接”后回 Dashboard，每个销售 Step 就能选择真实 Flow。

Token 等同账号钥匙：只允许项目负责人在本页亲自贴上，不要发进 WhatsApp 群、Google Docs 或 FAQ。保存成功后 Token 空格会自动清空，这是正常的。
