# AI 回复规则

## 默认分类

| 类型 | 处理 |
| --- | --- |
| 简单 FAQ | 可自动回复 |
| 看价前问题 | 回答后接下一步 Flow |
| 看价后异议 | 成交回复，建议审批 |
| 下单意图 | 收资料 / 提交订单 |
| Receipt | 审核或转人工 |
| 投诉 / 退款 / 健康敏感 | 转人工 |

## 自动回复边界

可以自动：

- 用法
- 多久见效
- 产品适合谁
- 简单价格说明
- 低风险 FAQ

不要自动：

- 医疗承诺
- 投诉
- 退款争议
- receipt 金额不对
- 客户很生气
- AI 信心低

## 学习闭环

Dashboard 批准后，可以点：

```text
以后类似 Case 自动通过
```

系统会记录：

- 顾客问题关键词
- AI 回复
- 人工是否改过
- 最终是否批准
- 风险等级

下一次类似 Case，AI 可以少问人工。
