# AI 回复规则

## 默认分类

| 类型 | 处理 |
| --- | --- |
| 广告关键词进线 | 交给 ChatDaddy Step 1，AI 不额外回复 |
| 顾客疑问 / 效果顾虑 | Dashboard 待批准，客服确认后再发 |
| 看价前普通问题 | 先人工确认，再决定是否接下一步 Flow |
| 看价后异议 | 成交回复，建议审批 |
| 下单意图 | 收资料 / 提交订单 |
| Receipt | 审核或转人工 |
| 投诉 / 退款 / 健康敏感 | 转人工 |

## 自动回复边界

可以自动：

- 顾客点击广告关键词后，由 ChatDaddy 自动进入 Step 1
- 已经设置好的固定 Follow-up Flow
- 已确认安全的系统流程动作

不要自动：

- 顾客问效果、用法、多久有效
- 顾客说试过没效、担心没效果、没有、考虑一下
- 医疗承诺
- 投诉
- 退款争议
- receipt 金额不对
- 客户很生气
- AI 信心低

## 学习闭环

Dashboard 批准后，可以点：

```text
以后类似情况少审核
```

系统会记录：

- 顾客问题关键词
- AI 回复
- 人工是否改过
- 最终是否批准
- 风险等级

下一次类似 Case，AI 可以少问人工。
